# Human setup

No secret or external configuration is required to implement or run automated tests.

For an optional live acceptance smoke test, the tester needs:
- a supported Codex CLI executable on PATH or configured in Hyper;
- a ChatGPT account permitted to use Codex;
- an available OS credential keyring;
- an interactive PowerShell 5.1 or PowerShell 7 Hyper profile.

The app itself must initiate browser sign-in. No API key or token is pasted into the repository.
